Combien de $1$ l'écriture binaire de $625$ contient-elle (exactement)?
%
\begin{enumerate}
  \item $6$
  \item $2$
  \item $5$
  \item $7$
  \item $4$
\end{enumerate}
%
\begin{proof}
Décomposons $625$ comme suit:
%
\begin{align}
  625 &= 512 + 113 \\
  &= 512 + 64 + 49 \\
  &= 512 + 64 + 32 + 16 + 1\\
  &= 2^9 + 2^6 + 2^5 + 2^4 + 2^0.
\end{align}
L'écriture binaire de $625$ comporte $5$ chiffres $1$ (exactement): réponse \textbf{C}.
\end{proof}
